## Installation

You can install react-crazy-hooks via bun, npm or yarn:

### Using bun

```bash
bun add react-crazy-ui
```

### Using npm

```bash
npm install react-crazy-ui
```

### Using yarn

```bash
yarn add react-crazy-ui
```
